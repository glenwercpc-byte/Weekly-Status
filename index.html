주일예배 출석현황

표시 방법 — 칸을 클릭: 빈칸(출석) → ✓ → X(결석) → 직접입력(예: 출타, 한국) → 다시 빈칸. 칸을 우클릭하면 바로 출석(빈칸)으로 초기화됩니다. 환우·타교·EM·타주·장결는 직접 수정하기 전까지 매주 자동으로 유지됩니다. ● 표시가 있는 이름을 클릭하면 남/여를 선택할 수 있고, 선택하면 해당 칸만 남고 나머지 칸은 화면에서 사라집니다 (다시 클릭해서 언제든 바꿀 수 있습니다).
이 표는 Google Sheets를 백엔드로 사용합니다 (Code.gs 참고). "저장 및 동기화" 버튼을 누르면 현재 화면 전체를 다시 한 번 확실하게 저장한 뒤, 서버의 최신 상태를 다시 불러옵니다 — 여러 관리자가 동시에 열어 작업할 때나, 개별 저장이 유실된 것 같을 때 눌러주세요.
첫 번째 날짜 칸은 이번 주(현재 진행 중인) 날짜이고, 두 번째 날짜 칸은 조회용입니다 — 조회용 칸에서 날짜를 골라 조회 버튼을 누르면 그 날짜의 기록을 불러와 볼 수 있습니다 (지난 기록은 편집이 불가능합니다 — "현재 주로 돌아가기"로 다시 편집 화면으로 돌아올 수 있습니다).
1~200번은 일반 교인(가나다순), 201번부터는 EM(영어이름) 교인 전용 구역(알파벳순)으로 서로 분리되어 관리됩니다. 편집 모드에서 빈 줄에 이름을 입력하거나 이름을 지우면 해당 구역만 자동으로 재정렬됩니다. EM 구역이 꽉 차면 자동으로 20자리가 더 늘어납니다.
자료 제출 메뉴의 "전체 결석자 명단"은 "새 주 시작"으로 보관된 지난 주차 기록(기록 시트)과 이번 주 현재 데이터를 기준으로 연속 결석 주수별 명단을, "샘터별 결석자 명단"은 이번 주 결석자를 샘터 번호순으로 보여줍니다.
